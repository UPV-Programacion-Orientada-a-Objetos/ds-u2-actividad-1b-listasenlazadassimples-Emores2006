\doxysubsection{Descripción detallada}
Lista enlazada simple para almacenar punteros a \doxylink{classSensorBase}{Sensor\+Base}. 

Permite gestionar de forma polimórfica diferentes tipos de sensores (\doxylink{classSensorTemperatura}{Sensor\+Temperatura}, \doxylink{classSensorPresion}{Sensor\+Presion}) a través de punteros a la clase base.
